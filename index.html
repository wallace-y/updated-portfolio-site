The Companies House Search web application is a project built with React that allows users to search for companies from the Companies House register. It was one of the first projects I built using React and a RESTful API.

The application uses the useState and useEffect hooks, along with several components. It is also styled using Bootstrap.

The App component manages an array of companies and a search term using useState and fetches data from the Companies House API using useEffect.

The searchCompanies function makes a request to the API and sets the companies state variable to the result. The component renders a search input, a search button, and a list of CompanyCard components to display the search results or a message if no companies are found.

The Text Prompt Generator is an HTML webpage that generates a random text prompt for DALL-E, an AI image generation model by OpenAI, and shows an image gallery of previously generated images. The code uses the Bootstrap framework for styling and responsiveness.

There is a "Generate" button that calls a JavaScript function to generate the random prompt, and a prompt output box to display the generated prompt.

The image gallery section displays generated images in a card layout using Bootstrap's grid system. Each card contains an image and a title of the generated image.

The Words Of Affirmation project is a Node.js server that sends a daily email containing a random affirmation message to a designated email address.

The project is hosted via Render. As well as the server side daily affirmation, there is a front-end element that providers users with a randomly generated affirmation each time the site is loaded.

The server is set up to run a cron job to run every day at 7:45 am that sends an email to the designated email address using the nodemailer library. The email contains a message generated from the affirmations.js file.

I learn't a lot of new things in this project, notably seting up Cron and using dotenv for managing environment varibles, as well as obtaining more practice with Node.js and Express. This is also one of the first projects I have launched to a hosted server using Render.
